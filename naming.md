
##  Naming Convention for Variables
### Examples
    const dogName = 'Droopy';

### Examples

    let hasOwner = true;

## Naming Convention for Functions
### Examples

    function getName(dogName, ownerName) { 
      return '${dogName} ${ownerName}';
    }
